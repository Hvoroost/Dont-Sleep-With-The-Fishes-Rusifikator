## $\color{blue}{\text{Установка}}$

Выберите один из двух удобных способов установки русификатора:

### $\color{yellow}{\text{Вариант 1. Автоматическая установка (Рекомендуется)}}$
1. Перейдите во вкладку **Releases** справа и скачайте файл инсталлятора (`.exe`).
2. Запустите скачанный установщик.
3. Программа автоматически попытается найти папку с игрой в Steam. Если этого не произошло, нажмите «Обзор» и вручную укажите путь к корневой папке игры.
4. Дождитесь конца установки.
5. Запускайте игру.

### $\color{yellow}{\text{Вариант 2. Ручная установка}}$ 
1. Скачайте архив с файлами локализации.
2. Распакуйте его и скопируйте папку `DontSleepWithTheFishes_Data`.
3. Перенесите скопированную папку **в корень игры** (туда, где лежит файл `Dont Sleep With The Fishes.exe`).
4. Согласитесь на замену всех файлов.
5. Запускайте игру.
6. Если windows откажет в доступе следуйте следующим шагам:
   1. Откройте командую строку от имени администратора.
   2. Введите команду "copy [путь к папке DontSleepWithTheFishes_Data] [путь к папке игры]".
   3. Дождитесь конца копирования.
   4. Запускайте игру.
